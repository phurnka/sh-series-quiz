# SH Series Quiz 🎭

แบบทดสอบออนไลน์ "คุณเป็นตัวละครไหนใน SH Series?" ตามระบบนพลักษณ์ (Enneagram) ทั้ง 9 แบบ

## ✨ Features

- 7 คำถาม 4 ตัวเลือก
- ระบบคำนวณนพลักษณ์ 1–9
- ผลลัพธ์ 18 ตัวละคร แบ่งเป็น **เมะ** และ **เคะ**
- Dark mode รองรับอัตโนมัติ
- Responsive สำหรับมือถือ
- ปุ่มแชร์ผลลัพธ์ (Web Share API)

## 🚀 วิธีใช้งาน

### ลง Local

```bash
git clone https://github.com/YOUR_USERNAME/sh-series-quiz.git
cd sh-series-quiz
# เปิดไฟล์ index.html ในเบราว์เซอร์ได้เลย — ไม่ต้องติดตั้งอะไรเพิ่ม
open index.html
```

### Deploy บน GitHub Pages

1. Push โค้ดขึ้น GitHub repo
2. ไปที่ **Settings → Pages**
3. เลือก Source: `Deploy from a branch` → Branch: `main` → Folder: `/ (root)`
4. Save แล้วรอ 1–2 นาที
5. เข้าใช้งานที่ `https://YOUR_USERNAME.github.io/sh-series-quiz/`
